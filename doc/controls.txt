MachineLevelGuardian
--------------------------------------------------------------------------------
Description:
- This class ensures that an instance of itself with the same name can not be created on different processes on same machine.
- It is useful to ensure that a single instance of the application is running on the current machine.

ListInputControl
--------------------------------------------------------------------------------
Description:
- Reads a list of values from the Console input.
- Stops when an empty string is encountered.

Features:
- Optional label - Displays a label at the top of the list.
- Customizable bullet - The bullet that is displayed in front of each item.
- Customizable space amount after the bullet.
- Customizable items indentation.
- Generic control - Reads any type of data.
- Custom error message for type validation - When the value cannot be converted into the requested type.

ProgressSpinner
--------------------------------------------------------------------------------
Description:
- Displays a progress-like visual bar that moves continuously. It can be used for background jobs for which the remaining work cannot be predicted.

Features:
- Supports templates that control the aspect of the spinner (the displayed text for each frame).
- Can change the interval time of the frames.

Note: It does not support changing colors while spinning.

- Custom value range.

Table
--------------------------------------------------------------------------------
Description:
- Displays tabular data on columns and rows.

Features:
- Optional table title.
- Optional column headers.
- Optional border.
- Three types of borders out-of-the-box: plus-minus (+---+), single-line (┌───┐), double-line (╔═══╗)
- Custom borders.

Obs: when content is aligned to center and there is an odd number of spaces, the content is aligned slightly to the left.

Prompter
--------------------------------------------------------------------------------
Description:
- Displays a prompter and waits for the user to write a command.

Features:
- Read a single command or run in a loop, to read commands until the Stop method is called.
- Customizable prompter text.
- Customizable prompter glyph.
- Customizable amount of space before and after the prompter.
- EnsureBeginOfLine - If the cursor is not at the begin of the line when the prompter must be displayed, the prompter is displayed on the next line.

SelectableMenu
--------------------------------------------------------------------------------
Description:
A menu in which the user can navigate by using the up/down arrow keys.

Features:
- List display - Display the menu items vertically in a list.
- Up/Down navigation - Select a menu item using up/down keys.
- Shortcut key selection - Select a menu item using a shortcut key.
- Items horizontal alignment - Horizontally align the menu items inside the menu (Left, Center, Right).
- Enable/Disable menu items.
- Show/Hide menu items.
- Commands - Can attach Commands to menu items that will be executed when the menu item is selected.
- Menu horizontal alignment - Align the menu in the console (Left, Center, Right).
- Circular navigation - When reaching the top item go to the bottom item. When reaching the bottom item go to the top item.

TextMenu
--------------------------------------------------------------------------------
Displays a list of items and asks the user to select one by typing its id.
When a item is selected, if it has an associated command, it is executed.
Alternatively, if there is no Command associated with the item, the selected item can be retrieved and some decisions can be taken based on it.


--------------------------------------------------------------------------------
Console MVC
--------------------------------------------------------------------------------
- Provides an easy way to implement execution flows for each command.
- Configurable CommandProvider (the module used to receive commands from the user)
	- can be a prompter where the user types a command.
	- can be a menu from which the user selects an item.
- Routing table - to associate a command text with a controller (execution flow).
- Optional service provider - If none is provided, the controllers must have parameterless constructors.
	Exceptions: Without a service provider, the controller's constructors can still have as parameters the ConsoleApplication, the ICommandProvider and the ICommand.
