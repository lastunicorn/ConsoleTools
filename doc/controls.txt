Guardian
--------------------------------------------------------------------------------
Description:
- This class ensures that an instance of itself with the same name can not be created twice.
- It is useful to ensure that a single instance of the application is running.

ListInputControl
--------------------------------------------------------------------------------
Description:
- Reads a list of items from the Console input.
- Stops when an empty string is encountered.

Features:
- Optional label - Displays a label at the top of the list.
- Customizable label separator - Displays a separator (default ':') after the label.
- Customizable bullet - The bullet that is displayed in front of each item.
- Customizable space amount after the bullet.
- Customizable items indentation.

TextInputControl
--------------------------------------------------------------------------------
Description:
- Reads a value from the Console input.
- The control is generic. It can be instantiated to read any type of value.

Features:
- Custom label - Displays a label at the top of the list.
- Custom label separator - Displays a separator (default ':') after the label.
- Custom space amount between label and value.
- Custom default value to be used if the user types Enter (string empty).
- Custom error message for transforming the input string into the requested type.

ProgressSpinner
--------------------------------------------------------------------------------
- Displays a progress-like visual bar that moves continuously. It can be used for background jobs for which the remaining work cannot be predicted.
- Supports templates that control the aspect of the spinner (the displayed text for each frame).
- Can change the interval time of the frames.

Note: It does not support changing colors while spinning.

ProgressBar
--------------------------------------------------------------------------------
- Displays a progress bar and the associated value.
- Custom Value positioning: left or right.
- Custom unit of measurement.
- Custom label text.
- Custom value range.
- Option available to allow values outside range.

Table
--------------------------------------------------------------------------------
- Displays tabular data on columns and rows.
- Optional table title.
- Optional column headers.
- Optional border.
- Three types of borders out-of-the-box: plus-minus (+---+), single-line (┌───┐), double-line (╔═══╗)
- Custom borders.

Obs: when content is aligned to center and there is an odd number of spaces, the content is aligned slightly to the left.

Prompter
--------------------------------------------------------------------------------
- Displays a prompter and waits for the user to write a command.
- Can read a single command and then stop or cand run in a loop, to read commands until the Stop method is called.
- Customizable prompter text.
- Customizable prompter glyph.
- Customizable amount of space before and after the prompter.
- EnsureBeginOfLine - If the curson is not at the begin of the line when the prompter must be displayed, the prompter is displayed on the next line.

SelectableMenu
--------------------------------------------------------------------------------
Displays a menu, asks the user to select an item by choosing it with up/down arrow keys and closes the menu.
The chosen item is exposed by the SelectedItem property.
Features:
    - List display - Displays the menu items vertically in a list.
    - Up/Down navigation - Selects a menu item using up/down keys.
    - Shortcut key selection - Selects a menu item using a shortcut key.
    - Items Horizontal alignment - Horizontally align the menu items inside the menu (Left, Center, Right).
    - Enable/Disable menu items.
    - Display/Hide menu items.

--------------------------------------------------------------------------------
TextMenu
--------------------------------------------------------------------------------
Displays a menu that asks the user to select an item by typing its id and executes the command associated with the item.
Alternatively, if there is no Command associated with the item, the selected item can be retrieved and some decisions can be taken based on it.


--------------------------------------------------------------------------------
Console MVC
--------------------------------------------------------------------------------
- Provides an easy way to implement execution flows for each command.
- Configurable CommandProvider (the module used to receive commands from the user)
	- can be a prompter where the user types a command.
	- can be a menu from which the user selects an item.
- Routing table - to associate a command text with a controller (execution flow).
- Optional dependency resolver - If none is provided, the controllers must have parameterless constructors.
	Exceptions: Without a dependency resolver, the controller's constructors can still have as parameters the ConsoleApplication, the ICommandProvider and the ICommand.
